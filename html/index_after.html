    • Html

      HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web. Ela estrutura o conteúdo na web usando elementos como cabeçalhos, parágrafos e links, permitindo que os navegadores exibam texto, imagens e multimídia de forma eficaz. HTML é a base do desenvolvimento web, essencial para a construção de sites.

    • Css

      CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Com o CSS, é possível definir cores, fontes, layouts e espaçamentos, tornando as páginas web visualmente atraentes e organizadas. Ele permite que os desenvolvedores separem o conteúdo da apresentação, facilitando a manutenção e o design responsivo. O CSS é fundamental para criar interfaces modernas e interativas na web.

    • JavaScript+NodeJs

      JavaScript (JS) é uma linguagem de programação amplamente utilizada para adicionar interatividade a páginas web. Com o Node.js, é possível executar JavaScript no servidor, permitindo o desenvolvimento de aplicações web completas. Node.js utiliza um modelo assíncrono e orientado a eventos, ideal para construir APIs e aplicações em tempo real. Juntas, JavaScript e Node.js permitem criar experiências ricas e dinâmicas, tanto no lado do cliente quanto no servidor.

    • MySql

      MySQL é um sistema de gerenciamento de banco de dados relacional amplamente utilizado para armazenar e gerenciar dados. Ele organiza informações em tabelas e permite consultas e manipulações através da linguagem SQL (Structured Query Language). Conhecido por sua eficiência e escalabilidade, MySQL é uma escolha comum para aplicações web e é parte do stack LAMP (Linux, Apache, MySQL, PHP/Python).
